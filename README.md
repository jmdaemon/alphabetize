# Alphabetize

Lexicographically orders lists of strings and imports.

## Usage

``` bash
$ alphabetize \
    "#include <pcre.h>" \
    "#include <stdlib.h>" \
    "#include <string.h>" \
    "#include <math.h>" \
    "#include <argp.h>" \
    "#include <stdio.h>" \
    "#include <stdbool.h>" \
```

## Installation

Run `make build` and `sudo make install` to install alphabetize to `/usr/local/bin/`
